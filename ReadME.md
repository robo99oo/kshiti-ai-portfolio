👩‍💻 About The Project

Kshiti AI Portfolio is an interactive AI-powered portfolio platform designed to go beyond traditional resumes.

Instead of viewing static PDFs, recruiters can:

interact with an AI recruiter assistant
explore enterprise AI projects
ask technical questions
use voice-assisted recruiter interaction
generate ATS-tailored resumes
understand measurable business impact through AI systems

This project focuses on production-style GenAI presentation using modern enterprise AI concepts and recruiter-centric design.

📦 Installation

Clone the repository:

git clone https://github.com/robo99oo/kshiti-ai-portfolio.git

Move into project folder:

cd kshiti-ai-portfolio

Install dependencies:

pip install -r requirements.txt

Run the project:

streamlit run app.py
